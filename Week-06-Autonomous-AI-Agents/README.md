## 🏗️ From Chat to Agent

```
CHATBOT (Weeks 1-5):              AGENT (Week 6):
┌──────────────────┐              ┌──────────────────────────────┐
│ User asks        │              │ User gives GOAL              │
│ → AI answers     │              │ → Agent PLANS steps          │
│ → Done           │              │ → Agent EXECUTES step 1      │
│                  │              │ → Agent OBSERVES result      │
│ Single step      │              │ → Agent PLANS next step      │
│ No autonomy      │              │ → Agent EXECUTES step 2      │
│ No tool use      │              │ → ... until GOAL achieved    │
└──────────────────┘              │                              │
                                  │ Multi-step, autonomous       │
                                  │ Uses tools, plan, reason     │
                                  └──────────────────────────────┘
```
